# CucumberFramework

A Java-based Cucumber BDD automation framework using Selenium and Maven.

## 🔧 Tech Stack
- Java
- Cucumber
- Selenium WebDriver
- Maven
- JUnit
- WebDriverManager
- VS Code

## 📂 Project Structure
src/
├── main/java/com/test
│ ├── PageObject/
│ └── Utilities/
├── test/java/com/test
│ ├── StepDefinition/
│ └── TestRunner/
└── test/resources/Features/
